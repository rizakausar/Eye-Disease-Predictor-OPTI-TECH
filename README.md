# Eye Disease Predictor (OPTI-TECH)

This is an AI-based web application built with **Flask** and **CNN** (TensorFlow) that predicts eye diseases from uploaded images.

## 🚀 Features
- **User Authentication:** Secure login and registration.
- **Disease Prediction:** Upload an eye image to get an instant diagnosis.
- **Database:** Stores user records using SQLite/SQLAlchemy.

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **Frontend:** HTML, CSS
- **Machine Learning:** TensorFlow, Keras
- **Database:** SQLAlchemy
